# Asset Guide

Place production assets under `public/images/`.

Recommended folders:

```txt
public/images/motion/{slug}/
public/images/illustration/{slug}/
public/images/interactive/{slug}/
public/images/journal/{slug}/
public/logos/
public/og/
public/icons/
```

Recommended dimensions:

- Project thumbnail: 1600 x 1200, 4:3
- Journal thumbnail: 1600 x 1200, 4:3
- Motion poster: 1920 x 1080, 16:9
- Project hero: 2400 x 1350, 16:9
- Portrait illustration: 1600 x 2000, 4:5
- Landscape illustration: 2000 x 1500, 4:3
- Open Graph: 1200 x 630

Use lowercase kebab-case file names.
