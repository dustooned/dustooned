# Design System

Global styles live in `src/styles/`.

Files:

- `tokens.css`: colors, type, spacing, radius, shadows
- `layout.css`: shells, page structure, prose, galleries
- `nav.css`: desktop and mobile navigation
- `cards.css`: project, journal, and gateway cards
- `lightbox.css`: Motion video lightbox
- `global.css`: imports and base element styling

Cards use a shared 4:3 media area and consistent metadata/body structure.
